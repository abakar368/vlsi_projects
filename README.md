# VLSI Projects Portfolio

A collection of portfolio-quality RTL design and verification projects implemented in Verilog HDL. This repository showcases larger digital hardware designs with clean, parameterized RTL and professional self-checking verification methodologies.

## Repository Goals

This repository is dedicated to building industry-style RTL design projects that demonstrate:

- RTL Design
- Parameterized Hardware Design
- Functional Verification
- Self-Checking Testbenches
- Directed and Random Testing
- Clean and Synthesizable Verilog HDL
- Simulation using QuestaSim and Icarus Verilog

## Current Projects

| Project | Description | Status |
|---------|-------------|:------:|
| Parameterized N-bit ALU | Arithmetic, logical, and shift operations with Carry, Overflow, Negative, and Zero flags, verified using directed and self-checking random testbenches. | ✅ |

## Verification Methodology

Projects in this repository follow a structured verification methodology using:

- Golden Reference Models
- `compute_expected()`
- `run_test()`
- `check_results()`
- Directed Test Cases
- Randomized Testing
- Automatic Result Checking
